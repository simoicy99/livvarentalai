## Project Context

Livva is a startup that automates long-term rentals for landlords and tenants.

Core idea:
- Intelligent agents handle the rental workflow instead of humans doing everything manually.
- Agents can:
  - Post and update listings across multiple platforms
  - Search for and match tenants and landlords
  - Handle messaging and coordination
  - Use a shared trust network so verified information about tenants / landlords becomes reusable “reputation data”
- The vision: make renting faster, fairer, and more human with AI.

High-level product:
- Multi-channel listing + sync
- Matchmaking between rooms / units and tenants
- Unified communication layer (messages, inquiries, follow-ups)
- “Trust network” layer (scores, verification, history)
- Payment & deposit flow (likely Stripe Connect, with things like “hold for 1 day then release to landlord” eventually)
- Landlord dashboard + tenant dashboard
- Admin tools for the Livva team

This project is part of a broader **“rental ai project” / Livva** codebase that the founder is actively building.

When writing code:

1. **Be explicit and complete**
   - Provide full, copy-pasteable files.

3. **Explain briefly**

     - What you implemented

4. **UI / UX**

5. **Error handling & realism**
